# Bundled Fonts

This project bundles the following font files in the `fonts/` folder for in-game rendering:

- `DejaVuSerif-Bold.ttf` (display headings)
- `LiberationSerif-Regular.ttf` (body and card text)
- `DejaVuSansMono.ttf` (fallback monospaced font)

## License Notes

- DejaVu font family is distributed under a free/open license based on Bitstream Vera with DejaVu additions.
- Liberation font family is distributed under the SIL Open Font License 1.1.

These files are included to avoid relying on system fonts and to keep typography consistent across platforms.
